Site Plan: Personal Dashboard

Site Name

Personal Dashboard

This name reflects a customizable space where I can manage personal information, such as bookmarks, notes, and weather updates

Site Purpose

The site serves as a central hub for personal organization, allowing to access my bookmarks, view weather data, and take notes

Scenarios

  • How can I quickly access my favorite websites?
  • What is the weather forecast for today?
  • Where can I jot down quick notes or reminders?

Color Schema

  • Black (#040316): Used for text
  • Primary (#2f27ce): Used for visual elements to pop
  • Accent (#443dff): Used for accenting elements

Typography

  • Headings: Inter, sans-serif (used for h1 and h2).
  • Body Text: Roboto, sans-serif (used for paragraphs).
